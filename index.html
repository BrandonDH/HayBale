Mail Bale

Format bulk email lists for pgAdmin and PostgreSQL.

I made this script to take a list of email addresses and format them so they're ready to be consumed by PgAdmin.

Fork Mail Bale
Input a list like this:
          
  user1@user1.com
  user2@user2.com
  user3@user3.com
Output a list like this:
          
  {user1@user1.com, user2@user2.com, user3@user3.com}
